布尔恒等式<br>
| **类别**         | **恒等式**                                                                 |
|------------------|---------------------------------------------------------------------------|
| **恒等律**       | \( A + 0 = A \)                                                           |
|                  | \( A • 1 = A \)                                                       |
| **零元律**       | \( A + 1 = 1 \)                                                           |
|                  | \( A • 0 = 0 \)                                                       |
| **幂等律**       | \( A + A = A \)                                                           |
|                  | \( A • A = A \)                                                       |
| **互补律**       | \( A + A' = 1 \)                                                |
|                  | \( A • A' = 0 \)                                            |
| **双重否定律**   | \((A')' = A \)                                         |
| **交换律**       | \( A + B = B + A \)                                                       |
|                  | \( A • B = B • A \)                                               |
| **结合律**       | \( A + (B + C) = (A + B) + C \)                                           |
|                  | \( A • (B • C) = (A • B) • C \)                           |
| **分配律**       | \( A • (B + C) = A • B + A • C \)                             |


De'Morgan's Law<br>
德摩根定理(ド・モルガンの法則)
| **形式**         | **布尔表达式**                                                                 |
|------------------|------------------------------------------------------------------------------|
| **基本形式**     | \( (A + B)' = A' • B' \)                     |
|                  | \(A • B)' = A' + B' \)                     |
| **逻辑门形式**   | **与非门（NAND）**：\( \(A • B)' = A' + B'\) |
    |                  | **或非门（NOR）**：\( (A + B)' = A' • B' \) |
